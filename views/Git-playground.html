    Local Set Up

    1. Open your command line/terminal.
    2. Use the cd command to navigate to your folder. For exampple cd desktop/projectfolder
    3. Type in git init. This will set up a repository(repo) for you locally.
    4. A message like this should pop up: "Initialized empy git reporsitoryv in dir/filename/stuff/here"

    Local Commits

    1. cd to the proper directory in the command line.
    2. enter git add . Note: This will add all the files that you've changed.
    3. enter git commit -m "A note about the things you've changed"Note: The words in quotes can be anything
    4. Your code us now backed up locally
